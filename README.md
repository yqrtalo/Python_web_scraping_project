# 2006-2025 全球電影票房數據分析

## 1. 專題概述
* **選題動機**：鑑於 2026 年將上映 MCU 的重點影視作品，其票房影響力是否具延續性，激發了我對大 IP 電影對票房影響力的好奇。
* **資料來源**：利用 Python 爬蟲技術自 **Box Office Mojo** 與 **IMDb** 抓取 2006 至 2025 年之原始數據。
* **資料範圍與欄位**：
    * **時間範圍**：2006 年至 2025 年。
    * **對象**：年度全球票房 Top 50 作品。
    * **主要欄位**：年份、排名、片名（中/英文）、全球票房、IMDb ID、電影類型、是否屬於 MCU。
* **關鍵套件**：`Requests`、`BeautifulSoup`、`Pandas`、`Matplotlib`。

## 2. 分析圖表說明
本專案產出三張核心分析圖表，透過不同維度解析市場：
1. **全球票房 Top 50 結構分析**：觀察 Top 1-10、Top 11-30、Top 31-50 作為「市場火車頭」的產值貢獻與佔比變化。
2. **年度票房集中度趨勢圖**：聚焦 Top 5和TOP6-20 觀察現象級大片對市場的壟斷力及競爭格局。
3. **MCU非MCU年度票房對比圖**：追蹤自 2008 年《鋼鐵人》起， MCU票房和非MCU票房的對比。

## 3. 技術困難與克服方案
在資料採集與處理過程中，克服了以下技術難點：
* **資料收集限制**：因 Box Office Mojo 與 IMDb 未提供官方 API，全面採用網路爬蟲進行數據自動化抓取。
* **動態網頁反爬機制**：
    * **困難**：網站設有反爬機制，頻繁抓取易遭封鎖。
    * **解決**：在爬蟲邏輯中加入隨機延遲（Random Sleep）與隨機額外大區間延遲，模擬真人瀏覽行為，確保連線穩定性。
* **網頁解析障礙**：
    * **困難**：使用 `BeautifulSoup` 分析 IMDb 網頁時，部分瀏覽器可見的資料欄位無法直接抓取。
    * **解決**：重新比對網頁源碼結構，找出資料渲染路徑，並針對特定標籤進行深度解析。

## 4. 如何執行
1. 安裝必要套件：`pip install -r requirements.txt`
2. 執行主程式抓取資料：`python main.py`
3. 執行更新程式補全資訊：`python update.py`
4. 產出分析圖表：`python analysis_utils.py`
